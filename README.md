# Cosmic Chaos

우주를 탐험하는 방치형 클리커 게임

## 🎮 게임 소개

Cosmic Clicker는 우주를 배경으로 한 증분형(Incremental) 방치형(Idle) 게임입니다. 
에너지를 수집하고, 생성기를 업그레이드하며, 다중 우주를 정복하세요!

## ✨ 주요 기능

- 🚀 **30개 이상의 생성기**: 수집 로봇부터 타임라인 작가까지
- 🌌 **5단계 환생 시스템**: 차원 도약, 시간 역행, 우주 승천, 공허 통합, 현실 조작
- 💎 **15개 이상의 유물**: 영구적인 버프 제공
- 🏆 **다양한 업적**: 게임 진행에 따른 보상
- 🤖 **자동화 시스템**: 생성기 및 업그레이드 자동 구매
- 💫 **황금 혜성**: 랜덤 보너스 이벤트

## 🎯 게임 방법

1. 화면 중앙의 에너지를 클릭하여 에너지 수집
2. 에너지로 생성기 구매 및 업그레이드
3. 충분한 에너지를 모아 환생(Prestige)하여 영구 버프 획득
4. 상위 환생 단계로 진행하며 더 강력한 보상 획득

## 🛠️ 기술 스택

- **Frontend**: React + Vite
- **Styling**: CSS3 (Glassmorphism Design)
- **State Management**: React Hooks
- **Storage**: LocalStorage
- **Monetization**: Google AdSense
